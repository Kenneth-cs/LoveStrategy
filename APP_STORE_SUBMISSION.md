## 🔐 第三步：隐私和合规

### 1. 隐私政策 URL（必需）
你需要一个公开可访问的隐私政策页面。

**选项 1：使用 GitHub Pages（免费）**
```bash
# 在你的 GitHub 仓库创建 docs 文件夹
mkdir docs
# 创建 privacy.html
# 在 GitHub 仓库设置中启用 GitHub Pages
# URL: https://kenneth-cs.github.io/LoveStrategy/privacy.html
```

**选项 2：使用第三方服务**
- Termly.io（免费）
- PrivacyPolicies.com（免费）

### 2. 用户协议 URL（可选）
同样方式创建 `terms.html`

### 3. 支持 URL（可选）
创建 `support.html` 或使用邮箱

### 4. 隐私实践问卷
App Store Connect 会问你：

**收集哪些数据？**
- ✅ 设备信息（用于崩溃报告）
- ✅ 使用数据（功能使用频率）
- ❌ 不收集个人身份信息
- ❌ 不收集联系方式
- ❌ 不收集位置信息

**数据用途？**
- ✅ 应用功能
- ✅ 分析
- ❌ 不用于广告
- ❌ 不出售给第三方

**第三方 SDK？**
- ✅ 字节跳动火山引擎（AI 分析）
  - 用途：图像识别和自然语言处理
  - 数据：聊天截图（临时处理，不永久保存）

## 🚀 第四步：App Store Connect 提交流程

### 1. 登录 App Store Connect
访问：https://appstoreconnect.apple.com

### 2. 创建新 App
1. 点击「我的 App」→「+」→「新建 App」
2. 填写信息：
   - **平台**: iOS
   - **名称**: 恋爱军师
   - **主要语言**: 简体中文
   - **套装 ID**: 创建新的（例如：com.yourname.lovestrategy）
   - **SKU**: lovestrategy（唯一标识符）
   - **用户访问权限**: 完全访问权限

### 3. 填写 App 信息

#### 3.1 App 信息
- **名称**: 恋爱军师
- **副标题**: AI情感分析·高情商回复助手
- **类别**:
  - 主要类别：生活（Lifestyle）
  - 次要类别：娱乐（Entertainment）

#### 3.2 定价和销售范围
- **价格**: 免费
- **销售范围**: 中国大陆（或全球）

#### 3.3 App 隐私
- 点击「管理」
- 填写隐私实践问卷（见上文）
- 添加隐私政策 URL

#### 3.4 年龄分级
回答问卷：
- **频繁/强烈的成人/性暗示主题**: 否
- **频繁/强烈的亵渎或低俗幽默**: 否
- **频繁/强烈的恐怖/惊悚主题**: 否
- 其他都选「否」
- 预计评级：**4+** 或 **12+**

### 4. 准备提交

#### 4.1 在 Xcode 中准备构建版本
```bash
1. 打开 Xcode
2. 选择 Product → Archive
3. 等待归档完成
4. 在 Organizer 中选择归档
5. 点击 Distribute App
6. 选择 App Store Connect
7. 选择 Upload
8. 等待上传完成（可能需要 10-30 分钟）
```

#### 4.2 在 App Store Connect 中创建版本
1. 进入你的 App
2. 点击「+版本或平台」
3. 选择「iOS」
4. 输入版本号：**1.0.0**

#### 4.3 填写版本信息
- **此版本的新增内容**: 填写更新说明（见上文）
- **宣传文本**: 填写宣传文本（见上文）
- **描述**: 填写 App 描述（见上文）
- **关键词**: 填写关键词（见上文）
- **支持 URL**: 你的支持页面 URL
- **营销 URL**: （可选）

#### 4.4 上传素材
1. **App 预览和截图**:
   - 上传 5-10 张截图
   - 上传预览视频（可选）

2. **App 图标**:
   - 系统会自动从构建版本中获取

#### 4.5 选择构建版本
1. 在「构建版本」部分点击「+」
2. 选择刚才上传的构建版本
3. 等待处理完成（可能需要几分钟）

#### 4.6 填写其他信息
- **版权**: © 2024 你的名字
- **App 审核信息**:
  - 联系人信息（你的邮箱和电话）
  - 审核备注：
    ```
    感谢审核！
    
    本应用是一款基于 AI 的情感分析工具：
    1. 用户上传聊天截图进行分析
    2. AI 提供情感洞察和回复建议
    3. 所有分析结果仅供娱乐参考
    
    测试账号：无需账号，直接使用
    测试说明：
    - 点击「鉴渣雷达」上传任意聊天截图
    - 点击「拿捏助手」输入任意文字生成回复
    - 点击「心理投射」上传截图查看卦象
    
    隐私说明：
    - 图片仅用于 AI 分析，不永久保存
    - 数据本地存储，不上传服务器
    
    如有任何问题，请随时联系。
    ```

### 5. 提交审核
1. 检查所有信息是否填写完整
2. 点击右上角「提交以供审核」
3. 确认提交

## ⏰ 第五步：等待审核

### 审核时间
- **通常**: 1-3 天
- **节假日**: 可能延长到 5-7 天
- **首次提交**: 可能需要更长时间

### 审核状态
- **等待审核**: 排队中
- **正在审核**: 审核中（通常 24-48 小时）
- **被拒绝**: 需要修改后重新提交
- **准备销售**: 审核通过，可以发布

### 常见拒绝原因及解决方案

#### 1. 隐私问题
**原因**: 隐私政策不完整或未说明数据使用
**解决**: 
- 完善隐私政策
- 在 App 中添加隐私说明
- 确保隐私政策 URL 可访问

#### 2. 功能不完整
**原因**: App 功能无法正常使用或缺少核心功能
**解决**:
- 确保所有功能都能正常工作
- 提供详细的测试说明
- 如果需要 API Key，在审核备注中说明

#### 3. 内容问题
**原因**: 包含不当内容或误导性描述
**解决**:
- 避免使用"渣男"等敏感词
- 强调"仅供娱乐参考"
- 不要承诺 100% 准确

#### 4. 设计问题
**原因**: UI 不符合 Apple 设计规范
**解决**:
- 确保适配所有 iPhone 尺寸
- 使用系统字体和标准控件
- 避免模仿系统 UI

#### 5. 性能问题
**原因**: App 崩溃或性能差
**解决**:
- 修复所有崩溃
- 优化性能
- 在真机上充分测试
