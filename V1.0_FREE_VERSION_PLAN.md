## 💡 关键决策

### 决策 1: 免费版要不要限制次数？
**建议**: ✅ 要限制（每天 3 次）
- **原因**: 控制 API 成本
- **策略**: 足够体验，但不能无限使用
- **备注**: V2.0 再放开

### 决策 2: 要不要做分享功能？
**建议**: ⭐ P2 优先级（时间够就做）
- **原因**: 分享能带来自然增长
- **策略**: 生成带水印的精美卡片

### 决策 3: 要不要适配 iPad？
**建议**: ❌ 暂不适配
- **原因**: 目标用户主要用手机
- **策略**: V2.0 再考虑
